    Django 설치하기

    장고 설치하기

    Windows 기준

    먼저 Python 가상환경(Virtual environment)을 설치한다. 가상환경은 프로젝트 기초 전부를 Python/Django와 분리해준다. 웹사이트가 변경되어도 개발 중인 것에 영향을 미치지 않는다.

    프로젝트 디렉토리 안에 가상환경 설치하기

    0. 디렉토리 만들기
``` 
    $ mkdir django
    $ cd django

``` 

1. myvenv라는 가상환경 만들기

```
    $ python -m venv myvenv
```

2. 가상환경 실행하기

```
    $ myvenv\Scripts\activate
```

    파이썬에 장고 설치하기

    세 가지 방법으로 장고를 설치할 수 있다.

        1. 공식 릴리즈
       $ python -m pip install Django
       
    2. 운영체제 에서 제공하는 Django
        ( https://docs.djangoproject.com/ko/3.2/misc/distributions/ )

    3. 최신 개발 버전
       $ git clone https://github.com/django/django.git
       $ python -m pip install -e django/

    여기서는 공식 릴리즈를 사용하기로 한다.

    0. 파이썬 버전 최신으로 업그레이드 하기

python3 -m pip install --upgrade pip

1. 공식 릴리즈 받기
python -m pip install Django

    참고: Django 공식 개발문서 : https://docs.djangoproject.com/ko/3.2/
